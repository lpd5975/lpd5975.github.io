Synesthesia: The Color of Music

By Liam Dougherty (lpd5975)

Overview

The purpose of this project is to add color to videos of musical performance wherein the color’s movement would be simulating how sound waves realistically travel through space.

Description

There are a few moving parts to this project that will need to be synthesized in order for it to be a completed whole. The three main categories are:

  • Sound Wave Propagation through Space
  • Color Matching to Musical Pitches
  • Video Integration of Above Algorithms

Goals

The stretch goal of this project is to be able to take any video of musical performance, run the program, and have digital effects of color flow from the source of the music. As the timeline for the project is quite limited, the minimum viable product (and what the final presentation will entail) will be a single video of a singer with the color added. Traditionally, music is an extremely aural experience. The great benefit of this project would be the opportunity to combine the visual and aural arts in a completely new way.

Responsibilities

As this is a solo project, the three main categories of work described above will fall on myself.
